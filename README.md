# Python Calculator

A simple calculator built using Python.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ❌ Division by zero handling

## Technologies Used
- Python 3

## How to Run
1. Download the project.
2. Open a terminal.
3. Run:
   python calculator.py
